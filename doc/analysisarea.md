Analysis Area
=============

Default Analysis Area Layers
----------------------------

WFIPS comes with a default set of layers to choose popular analysis areas.  The
layers are typical administration boundaries for some government agencies:

- GACC
- USFS Forests
- USFS Districts
- US States
- US Counties
- Legacy FPU boundaries

--------------------
